# task-manager-lite
Aplicação REST criada com Java 17 + Spring Boot 3 para demonstrar domínio em desenvolvimento backend. O projeto implementa um gerenciador de tarefas com CRUD completo, validações, conversão DTO ↔ entity, H2 Database, exceções personalizadas e boas práticas de arquitetura. Ideal para estudos, entrevistas técnicas e construção de portfólio backend.
